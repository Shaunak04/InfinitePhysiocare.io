# Infinite PhysioCare

GitHub Pages-ready website for a physiotherapy and rehabilitation clinic.

## What is included
- SEO-focused homepage with clinic story, services, leadership, hours, and booking CTA
- Dedicated pages for Book appointment, Meet your experts, Success stories, About us, and Contact info
- Static blog library with three starter posts
- Google Form appointment page with a calendar-sync workflow placeholder
- `robots.txt`, `sitemap.xml`, `404.html`, `site.webmanifest`
- Custom domain support for `infinitephysiocare.org`

## Booking setup
1. Create your Google Form.
2. Replace the placeholder form URL on `book-appointment.html`.
3. Connect responses to Google Sheets.
4. Use Google Calendar appointment scheduling or your preferred Sheets-to-Calendar automation.

## Deployment
1. Push the files to a public GitHub repository.
2. Enable GitHub Pages from the `main` branch.
3. Set the custom domain to `infinitephysiocare.org`.
4. Add the DNS records required by GitHub Pages.

## Files to edit first
- `book-appointment.html` — replace the Google Form link
- `contact.html` — confirm phone, email, Instagram, and address
- `experts.html` — add real doctor/team details if needed
- `stories.html` — replace placeholder testimonial copy
- `blogs/*.html` — add new clinic articles whenever you publish
